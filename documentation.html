Python Documentation

Welcome to the Python documentation. This guide will help you understand Python's features and how to use them effectively in your projects.

Getting Started

Python is a high-level, interpreted programming language known for its readability and versatility. To start using Python, you need to install it on your system.

Visit the official Python website at python.org to download the latest version for your operating system.

After installation, you can verify it by opening a terminal or command prompt and typing:

python --version

Python Syntax

Python syntax is designed to be readable and straightforward. Unlike many programming languages, Python uses indentation to define code blocks.

# This is a comment if True: print("This is indented") if True: print("This is indented more") print("Back to the first indentation level") print("No indentation")

Python statements typically end with a newline, though they can be extended with a backslash or within parentheses, brackets, or braces.

Data Types

Python has several built-in data types:

  • int - Integer values like 42, -7
  • float - Floating-point numbers like 3.14, -0.001
  • str - Text strings like "Hello, World!"
  • bool - Boolean values (True or False)
  • list - Ordered, mutable collections like [1, 2, 3]
  • tuple - Ordered, immutable collections like (1, 2, 3)
  • dict - Key-value pairs like {"name": "John", "age": 30}
  • set - Unordered collections of unique items like {1, 2, 3}

Control Flow

Control flow statements determine the execution order of your code:

# If-else statement x = 10 if x > 5: print("x is greater than 5") elif x == 5: print("x is equal to 5") else: print("x is less than 5") # For loop for i in range(5): print(i) # Prints 0, 1, 2, 3, 4 # While loop count = 0 while count < 5: print(count) count += 1
